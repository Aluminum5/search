Ahmia blacklists sites containing child abuse material from its index. This list contains a masked version of all hidden service .onion addresses that have been added to the Ahmia blacklist, and is updated whenever Ahmia blacklists a new .onion site for abuse material.

Blacklist information

Hidden service names are masked by creating an MD5 sum of the onion name plus a new line. For example, if the onion is abcdefghijk.onion, the resulting hash would be e271dc47fa80ddc9e6590042ad9ed2b7. This way it is possible for researchers and administrators to check if a site has been blacklisted from the index without making the .onion hostnames easily accessible.

Ahmia offers an API endpoint for this blacklisting. To check if a site has been blacklisted, make a GET request to https://ahmia.fi/address/$ONION_SERVICE_NAME/, where $ONION_SERVICE_NAME is the "abcdefghijk" part of the .onion address. If you have JavaScript enabled, the form below will perform a client side confirmation of being blacklisted or not.

Blacklist hash data is available for raw text download here. You may use this in your own search indexes to remove abusive content.
